## `dompetku`

## Manual Build 

> Download the code 

```bash
$ git clone https://github.com/AHyphaG/dompetku.git
$ cd dompetku
```
> Start the app <br/>
> For development
```bash
$ npm run dev
```

At this point, the app runs at `http://127.0.0.1:5500/`.

Unfortunately DATABASE is not yet shareable due to construction during development.
